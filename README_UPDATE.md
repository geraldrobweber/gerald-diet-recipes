# GitHub Update: Skyr-Frühstücke + Grill-Salate

Dieses Paket enthält:
- `index.html` als aktualisierte Startseite
- `rezept/fruehstueck/*.html` mit je einem Skyr-Obst-Rezept
- `rezept/salate/*.html` mit je einem Grill-Salat

Upload in GitHub:
1. ZIP entpacken.
2. In GitHub im Repository `gerald-diet-recipes` auf `Add file` → `Upload files`.
3. `index.html` ins Hauptverzeichnis hochladen und die bestehende Datei ersetzen.
4. Den ganzen Ordner `rezept` hochladen.
5. `Commit changes` klicken.
6. Nach dem GitHub-Pages-Deployment testen.
