# Geralds Rezeptbibliothek v1 – Website für Choosy-Import

Inhalt:
- `index.html`: Übersicht über alle Rezepte
- `rezepte/*.html`: eine einzelne Webseite pro Rezept
- Jede Rezeptseite enthält Schema.org Recipe JSON-LD Markup.

So nutzt du es:
1. ZIP entpacken.
2. Ordner online stellen, z. B. bei Netlify, Vercel oder GitHub Pages.
3. Die öffentliche URL eines einzelnen Rezepts in Choosy als Weblink importieren/speichern.

Hinweis:
Choosy muss die Seite öffentlich über eine URL erreichen können. Lokale Dateien wie `file:///...` sind für App-Importe meistens nicht geeignet.
